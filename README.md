# BRIA RMBG 2.0

This is an implementation of [BRIA's RMBG-2.0](https://huggingface.co/briaai/RMBG-2.0) as a [Cog](https://github.com/replicate/cog) model.

## Basic Usage

To run a prediction:

    sudo cog predict -i image=@giraffe.jpg
